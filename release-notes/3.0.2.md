# PhoneBlock 3.0.2

*2026-05-23*

## API

- Faster spam lookups: cover-indexed `NUMBERS_AGGREGATION_10/100` hash queries and a forced SHA1 index for prefix checks (#329)
- `/api/check-prefix` instrumented with per-phase latency timing (#329)
- OpenAPI spec explains wildcard votes and range detection

## Web App

- Proof-of-work is now required only for number lookups and the statistics page, not for the rest of the site
- Homepage tile links directly to the three landline setup options
- Richer social link previews: localized `og:locale` and large-image summary cards; dropped the legacy `keywords` meta tag

## Answer Bot

- Dropped the default `*` test-prefix that doubled responses when combined with accept-local
- Documented the `ACCEPT_LOCAL` interaction with Fritz!Box group calls

## Database

Schema version 26. Migrations 25–26 add covering indexes for the SHA1 hash lookups.
