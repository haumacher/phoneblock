# Locales to publish localized dongle assets for (issue #460).

# One locale code per line — the ui_lang value the device stores / the web UI
# selects (must satisfy config_lang_code_valid(): [a-z0-9-], 2..11 chars).
# The set matches the languages phoneblock.net supports (same codes as the
# mobile app's translateArb); `de` is the source (inline UI + mail_de.arb).
#
# All translations are authored during development and committed to git:
#   audio/announcement-<code>.alaw   (hand-recorded / synthesized)
#   l10n/mail/mail_<code>.arb        (de = the source; others translated)
#   l10n/ui/ui_<code>.arb            (de regenerated from index.html)
# A locale with a missing asset degrades gracefully (silent audio / German
# text). The release script only assembles committed files — no translation
# runs at release time. Add a language by adding it to targetLangs in
# l10n/mail/build.gradle + l10n/ui/build.gradle, translating, and listing it
# here; no firmware change.
