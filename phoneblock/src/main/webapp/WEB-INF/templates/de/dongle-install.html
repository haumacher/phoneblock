Dongle im Browser flashen

Ein ESP32-Board, ein USB-Kabel, ein Klick — die aktuelle PhoneBlock-Firmware wird direkt aus diesem Browser auf deinen Dongle übertragen.

Dein Browser unterstützt Web Serial nicht. Web-Flashing läuft nur in Desktop-Chrome, Edge oder Opera. Auf Firefox, Safari, Android und iOS kannst du den Dongle stattdessen mit der ESP-IDF-Toolchain über USB flashen.

1. Hardware vorbereiten

  • ESP32-DevKit mit ESP-WROOM-32E (z. B. NodeMCU-32E) — siehe Hardware-Hinweise.
  • USB-Datenkabel (kein reines Ladekabel) zwischen Computer und Dongle.
  • Auf Windows evtl. den USB-Serial-Treiber installieren — meist CP210x (Silicon Labs) oder CH340/CH341 (WCH).
  • Auf Linux muss dein Benutzer in der Gruppe dialout sein (sudo usermod -aG dialout $USER, danach einmal aus- und wieder einloggen).

2. Flashen

Klicke auf den Knopf, wähle im Browser-Dialog deinen Dongle aus und warte, bis der Vorgang abgeschlossen ist. Bei einem Erstflash wird der Flash-Speicher gelöscht — vorhandene Konfiguration geht dabei verloren.

3. Nach dem Flashen

  1. Der Dongle startet neu und spannt das WLAN PhoneBlock-Dongle-XXXX auf.
  2. Mit Smartphone oder Notebook in dieses WLAN wechseln, der Browser öffnet die Setup-Seite automatisch (Captive Portal).
  3. Heim-WLAN auswählen, SIP-Zugangsdaten und PhoneBlock-Token eintragen — Details in den Dongle-Hinweisen.
