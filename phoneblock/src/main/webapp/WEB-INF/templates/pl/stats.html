Statystyki PhoneBlock

Tutaj możesz zobaczyć, jak rozwija się społeczność PhoneBlock.

Rozwój użytkownika w ciągu ostatnich 30 dni

		{
			"type": "line",
			"data": {
				"labels": ,
				"datasets": 
			},
			"options": {
				"responsive": true,
				"maintainAspectRatio": false,
				"scales": {
					"y": {
						"ticks": {
							"precision": 0
						}
					}
				}
			}
		}

Dzisiaj: 0 użytkowników ogółem, +0 od wczoraj.

Liczeni są użytkownicy zarejestrowani w PhoneBlock.

Połączenia, głosy i zapytania w ciągu ostatnich 30 dni

		{
			"type": "bar",
			"data": {
				"labels": ,
				"datasets": 
			},
			"options": {
				"responsive": true,
				"maintainAspectRatio": false,
				"scales": {
					"y": {
						"ticks": {
							"precision": 0
						}
					}
				}
			}
		}

Tylko połączenia, które zostały faktycznie odebrane przez automatyczną sekretarkę w chmurze lub klucz sprzętowy PhoneBlock, liczą się jako połączenia przechwycone. Połączenia, które zostały odrzucone tylko za pośrednictwem listy blokowania (np. na Fritz!Box) nie są tutaj uwzględniane.

Aktywne użycia w ciągu ostatnich 30 dni

		{
			"type": "line",
			"data": {
				"labels": ,
				"datasets": 
			},
			"options": {
				"responsive": true,
				"maintainAspectRatio": false,
				"scales": {
					"y": {
						"ticks": {
							"precision": 0
						}
					}
				}
			}
		}

Użycie jest uważane za aktywne, o ile regularnie pobiera dane z PhoneBlock. W przypadku Fritz!Box uwzględniane są tylko przypadki użycia, które zostały zarejestrowane po przejściu na klucze API.

Zablokowane numery według kraju

		{
			"type": "pie",
			"data": {
				"labels": ,
				"datasets": [{
					"data": ,
					"backgroundColor": 
				}]
			}
		}

Numer jest uważany za zablokowany, gdy tylko osiągnie co najmniej 4 głosy SPAM. Ponieważ starsze głosy z czasem tracą na wadze, numer może ponownie spaść z listy blokowanych, jeśli nie zostaną dodane żadne nowe wiadomości.
