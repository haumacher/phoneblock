Blinker donglen i nettleseren

Ett ESP32-kort, én USB-kabel, ett klikk - den gjeldende PhoneBlock-fastvaren overføres direkte fra denne nettleseren til donglen din.

Leseren din støtter ikke webseriell Webblinking kjører bare i desktop Chrome, Edge eller Opera. På Firefox, Safari, Android og iOS kan du blinke donglen med ESP-IDF toolchain via USB i stedet.

1. Klargjør maskinvaren

  • ESP32-DevKit med ESP-WROOM-32E (f.eks. NodeMCU-32E) - se Hardware notes.
  • USB-datakabel (ikke bare en ladekabel) mellom datamaskin og dongle.
  • På Windows kan det hende du må installere den serielle USB-driveren - vanligvis CP210x (Silicon Labs) eller CH340/CH341 (WCH).
  • På Linux må brukeren din være i gruppen dialout (sudo usermod -aG dialout $USER, og logg deretter ut og inn igjen én gang).

2. Blinker

Klikk på knappen, velg din dongle i nettleserdialogen, og vent til prosessen er fullført. Flash-minnet slettes under en første blits - all eksisterende konfigurasjon vil gå tapt.

3. Etter blinking

  1. Dongelen starter på nytt og åpner WLAN PhoneBlock-Dongle-XXXX.
  2. Bytt til dette WLAN med smarttelefonen eller den bærbare PC-en, og nettleseren åpner automatisk oppsettssiden (captive portal).
  3. Velg hjemme-WLAN, angi SIP-tilgangsdata og PhoneBlock-token - mer informasjon finner du i Dongle-instruksjoner.

Kildekoden til fastvaren: phoneblock-dongle på GitHub.
