				"responsive": true,

I dag: 0 brukere totalt, +0 siden i går.

Brukere som er registrert med PhoneBlock, telles med.

Bare anrop som faktisk ble besvart av skytelefonsvareren eller en PhoneBlock-dongle, teller som avlyttede anrop. Anrop som bare ble avvist via blokkeringslisten (f.eks. på Fritz!Box), er ikke inkludert her.

En bruk regnes som aktiv så lenge den regelmessig henter data fra PhoneBlock. For Fritz!Box er bare bruksområder som ble registrert etter overgangen til API-nøkler, inkludert.

Blokkerte numre etter land

		{
			"type": "pie",
			"data": {
				"labels": ,
				"datasets": [{
					"data": ,
					"backgroundColor": 
				}]
			}
		}

Et nummer anses som blokkert så snart det har nådd minst 4 SPAM-stemmer. Etter hvert som eldre stemmer mister vekt over tid, kan et nummer falle av blokkeringslisten igjen hvis det ikke legges til flere nye meldinger.
