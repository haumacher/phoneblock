查看已拦截和未接来电

PhoneBlock Mobile 可显示所有已拦截的垃圾邮件电话和来自未知号码的未接来电。

数据保护：不会记录来自联系人列表中号码的通话。应用程序只处理未知号码。

工作原理

自动屏蔽

被拦截的垃圾电话会自动显示在应用程序中。您将收到包含号码和 SPAM 分数的通知。

电话详情

点击通话记录中的条目查看详情：

  • 垃圾邮件得分：社区负面评价的数量
  • 时间：通话时间
  • 类别：垃圾邮件类型（如广告、欺诈、竞争等）
  • 评论：其他 PhoneBlock 用户的报告和经验

在这里，您还可以对号码进行评分，或将其添加到您的个人黑名单/白名单中。

拦截是如何进行的？

PhoneBlock Mobile 使用 Android 的 CallScreeningService 在来电铃声响起之前对来电进行筛选。如果号码在垃圾邮件数据库中，并且超过了您设置的阈值，则会自动阻止来电。
