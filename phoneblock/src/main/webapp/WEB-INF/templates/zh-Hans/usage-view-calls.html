查看已拦截和未接来电

PhoneBlock Mobile 会检查所有来自未知号码的来电，并显示哪些已被拦截，哪些已经接通。

重要： 仅检查未知号码！

来自联系人列表中号码的呼叫不会被检查，也不会出现在 PhoneBlock 呼叫列表中。该应用程序只处理未保存在电话簿中的号码。

拦截垃圾邮件电话

被拦截的垃圾邮件电话会显示红色符号和垃圾邮件分数。您将收到有关拦截的通知。

哪些号码会被当作垃圾邮件拦截？

垃圾邮件号码（将被阻止）

如果出现以下情况，该号码将被拦截为垃圾邮件：

  • 您在 PhoneBlock 数据库中的垃圾邮件得分较高
  • 垃圾邮件得分超过您设置的阈值
  • 她上了你的个人黑名单（无论得分高低）

结果：您的电话没有响铃，呼叫被自动拒接。

合法号码（允许通过）

在下列情况下，一个号码被认为是合法的

  • PhoneBlock 数据库中没有该功能
  • 垃圾邮件得分较低，低于您的阈值
  • 她在您的个人白名单上（永远不会被屏蔽）

结果：您的电话正常响铃，呼叫接通。

拦截是如何进行的？

PhoneBlock Mobile 使用 Android 的 CallScreeningService 在来电铃声响起之前对来电进行筛选。如果号码在垃圾邮件数据库中，并且超过了您设置的阈值，则会自动阻止来电。

报告通话记录中的垃圾邮件号码

如果您接到了一个陌生号码打来的电话，而该号码是合法的，但实际上是垃圾邮件，您可以直接从呼叫列表中进行报告：

什么时候应该报告号码？

  • 该号码尚未录入 PhoneBlock 数据库（垃圾邮件得分较低或没有得分）
  • 电话明显不受欢迎（广告、欺诈等）
  • 您希望向社会公布这个号码
  • 该号码今后将被自动屏蔽

收到报告后怎么办？

  • 该号码已自动添加到您的黑名单</x1
  • 您将受到保护，今后不会再接到这个号码打来的电话
  • 您的信息为 PhoneBlock 数据库做出了贡献
  • 其他用户可从您的评级中获益

指示：报告垃圾邮件号码

重要：

如果您将某个号码报告为垃圾邮件，该号码将被添加到您的个人黑名单中，并在今后始终被阻止。此决定优先于社区评级。
