自动拒绝不受欢迎的来电

您的电话号码是否也落入了不法地址商的手中，您是否经常接到恼人的电话，希望 "优化 "您的电费账单，给您提供太阳能系统的建议，宣布另一个大奖或提供其他 "福利"？那么PhoneBlock正是您所需要的！

PhoneBlock 如何工作？

社区黑名单

PhoneBlock 是由 PhoneBlock 社区维护的一份涉嫌垃圾邮件的电话号码列表。PhoneBlock应答机会接受来自阻止列表上的来电者的呼叫，因此您的电话在接到这些呼叫时会保持静音。

替代品：FRITZ!Box 电话簿

或者，您也可以将黑名单作为电话簿导入 "FRITZ！Box "互联网路由器，并阻止来自专用电话簿中号码的呼叫。您不必再手动阻止号码范围，因为您的 "Fritz！Box "每晚都会自动更新阻止列表。一旦某个号码被添加到黑名单中，FRITZ！Box 就会自动拒绝该号码的来电。不过，应答机是更好的解决方案，因为电话簿的容量有限，无法容纳阻止列表中的所有号码。

报告垃圾邮件号码

如果由于该号码尚未被添加到拦截列表中，您仍然会接到不需要的电话，您可以报告该号码，从而保护整个 PhoneBlock 社区免受 更多垃圾电话的骚扰。您有几种选择：直接从 PhoneBlock 应用程序报告垃圾邮件，在网站上搜索号码并对其进行评分，或者通过在 PhoneBlock 地址簿中创建条目在您的 FRITZ!Box 中阻止该号码。一旦号码被举报，它就会被添加到拦截列表中，所有其他 PhoneBlock 用户都会受到自动保护。

有哪些要求？

PhoneBlock 支持多种平台，可拦截骚扰电话：

FRITZ!Box

使用 AVM 的 FRITZ!Box Internet 路由器阻止座机来电。或者安装 PhoneBlock 应答机，或者将拦截列表导入 电话簿。

安卓

使用 PhoneBlock 应用程序直接在您的 Android 手机上阻止垃圾电话。

iPhone

在您的 iPhone 中加入 PhoneBlock 封锁列表。

电话系统

将 PhoneBlock 集成到您的 电话系统中，作为屏蔽列表。
