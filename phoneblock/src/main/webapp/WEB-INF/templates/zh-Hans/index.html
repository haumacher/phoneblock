选择您的平台

盒子

建议：安装 PhoneBlock 应答机作为 VoIP 设备。电话答录机会自动接受垃圾电话，并与来电者进行毫无意义的对话，您的电话甚至不会响起。

替代方法：将被阻止的列表作为 电话簿导入您的 FRITZ!Box from AVM。来电者将被直接拒接（注意：电话簿大小有限，因此不会记录所有号码）。

安卓

使用 PhoneBlock 应用程序直接在您的 Android 手机上阻止垃圾电话。该应用程序会实时检查来电，并自动阻止已知的垃圾号码。您可以自行设置一个号码应阻止多少负面信息。

功能：通话记录、号码报告和评级、自有拦截列表、无需授权即可读取联系人信息。

iPhone

将 PhoneBlock blocklist 包含在 iPhone 通讯录中，当来电者出现在 blocklist 中时，iOS 会自动显示出来。设置通过 CardDAV 完成，iPhone 会自动更新列表。

电话系统

将 PhoneBlock 集成到您的电话系统或 PBX 中。可下载各种格式的区块列表，或通过 API 进行查询。是公司和专业电话系统的理想之选。

社区带来不同

强强联手：

任何人都可以加入！直接在应用程序中、网站上或通过在您的 FRITZ！Box中拦截来举报垃圾号码。一旦号码被举报，它就会被添加到所有用户的拦截列表中。参与的用户越多，我们就能更好地保护自己免受垃圾电话的侵害。
