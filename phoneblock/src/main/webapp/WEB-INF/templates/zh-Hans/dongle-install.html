在浏览器中闪烁加密狗

一块 ESP32 板、一根 USB 电缆、一次点击--当前的 PhoneBlock 固件将直接从浏览器传输到您的加密狗。

您的浏览器不支持网络串行网络闪存只能在桌面Chrome、Edge或Opera中运行。在 Firefox、Safari、Android 和 iOS 上，您可以使用 ESP-IDF 工具链通过 USB 闪存加密狗。

1. 准备硬件

  • 带有 ESP-WROOM-32E 的 ESP32-DevKit （例如 NodeMCU-32E） - 参见 硬件说明。
  • 电脑和加密狗之间的 USB 数据线（不只是充电线）。
  • 在 Windows 上，您可能需要安装 USB 串行驱动程序 - 通常是 CP210x (Silicon Labs) 或 CH340/CH341 (WCH)。
  • 在 Linux 上，用户必须在 dialout 组中（sudo usermod -aG dialout $USER，然后退出并重新登录一次）。

2. 闪烁

单击该按钮，在浏览器对话框中选择加密狗，然后等待该过程完成。闪存在初始闪存过程中会被删除，任何现有配置都将丢失。

3. 闪烁后

  1. 加密狗重新启动并打开 WLAN PhoneBlock-Dongle-XXXX。
  2. 使用智能手机或笔记本电脑切换到该 WLAN，浏览器会自动打开设置页面（专属门户）。
  3. 选择家庭 WLAN，输入 SIP 接入数据和 PhoneBlock 令牌 - 详见加密狗说明。

固件源代码： GitHub 上的 phoneblock-dongle.
