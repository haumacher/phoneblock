Flasher le dongle dans le navigateur

Une carte ESP32, un câble USB, un clic - le firmware PhoneBlock actuel est transféré directement sur ton dongle depuis ce navigateur.

Ton navigateur ne supporte pas Web Serial. Le flashage web ne fonctionne que dans Chrome, Edge ou Opera de bureau. Sur Firefox, Safari, Android et iOS, tu peux flasher le dongle à la place avec le toolchain EPI-IDF via USB.

1. préparer le matériel

  • DevKit ESP32 avec ESP-WROOM-32E (par ex. NodeMCU-32E) - voir Informations sur le matériel.
  • Câble USBde données (pas un simple câble de chargement) entre l'ordinateur et le dongle.
  • Sur Windows, installer éventuellement le pilote USB série - généralement CP210x (Silicon Labs) ou CH340/CH341 (WCH).
  • Sur Linux, ton utilisateur doit être dans le groupe dialout (sudo usermod -aG dialout $USER, puis se déconnecter et se reconnecter une fois).

2. flasher

Clique sur le bouton, sélectionne ton dongle dans la boîte de dialogue du navigateur et attends que le processus soit terminé. Lors du premier flash, la mémoire flash est effacée - la configuration existante est perdue.

3. après le flashage

  1. Le dongle redémarre et se connecte au WLAN PhoneBlock-Dongle-XXXX.
  2. Se connecter à ce WLAN avec un smartphone ou un ordinateur portable, le navigateur ouvre automatiquement la page de configuration (portail captif).
  3. Sélectionner le réseau local sans fil à domicile, entrer les données d'accès SIP et le jeton PhoneBlock - détails dans les Instructions relatives au dongle.
