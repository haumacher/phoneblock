Consulter les appels bloqués et manqués

PhoneBlock Mobile t'indique tous les appels SPAM bloqués et les appels manqués provenant de numéros inconnus.

Protection des données: Les appels provenant de numéros figurant dans ta liste de contacts ne sont pas enregistrés. L'application ne traite que les numéros inconnus.

Comment ça marche

Blocage automatique

Les appels SPAM bloqués sont automatiquement affichés dans l'application. Tu reçois une notification avec le numéro et le score SPAM.

Détails de l'appel

Appuie sur une entrée de l'historique des appels pour en voir les détails :

  • SPAM-Score: Nombre d'évaluations négatives de la communauté
  • Temps: Quand l'appel a eu lieu
  • Catégories: Type de SPAM (par ex. publicité, fraude, jeu-concours)
  • Commentaires: Témoignages et expériences d'autres utilisateurs de PhoneBlock

De là, tu peux aussi évaluer toi-même le numéro ou l'ajouter à ta liste noire/liste blanche personnelle.

Comment fonctionne le blocage ?

PhoneBlock Mobile utilise le CallScreeningService d'Android pour vérifier les appels entrants avant même qu'ils ne sonnent. Si le numéro se trouve dans la base de données SPAM et que la valeur seuil que tu as définie est dépassée, l'appel est automatiquement bloqué.
