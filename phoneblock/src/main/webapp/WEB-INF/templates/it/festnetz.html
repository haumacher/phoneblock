Quale soluzione vi si addice?

  • Dongle - se volete una difesa ottimale dallo SPAM con la migliore protezione dei dati, indipendentemente dal modello di router e dal provider.
  • Segreteria telefonica cloud - se si possiede una propria Fritz!Box (non una casella a noleggio) e si accetta che tutte le chiamate vengano controllate dal server PhoneBlock per una difesa ottimale dallo SPAM.
  • Rubrica dei blocchi - se avete una Fritz!Box (anche a noleggio) e non vi dispiace se ogni tanto vi sfugge uno spammer che è effettivamente nella lista dei blocchi.
