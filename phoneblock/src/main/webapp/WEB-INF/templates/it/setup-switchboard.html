Installazione nei sistemi telefonici

PhoneBlock può anche essere integrato nei sistemi telefonici. Poiché un sistema telefonico tende ad essere usato per scopi commerciali, si prega di notare le Note riguardanti l'uso commerciale di PhoneBlock, in particolare il disclaimer. Gli script per l'integrazione del sistema telefonico sono contributi della comunità di PhoneBlock. Pertanto, in caso di domande/problemi, si prega di contattare direttamente il rispettivo sviluppatore.

3CX

Per utilizzare PhoneBlock nel sistema telefonico di 3CX, date un'occhiata al progetto phoneblock-for-3cx.
