Search by number

You can look up any phone number in the PhoneBlock database, even without a call having taken place.

How to search for a number

Step 1: Open search

Tap on the magnifying glass icon at the top right to open the search function

Step 2: Enter number

Enter the phone number and tap on "Search"

What the search shows

After the search you will see the following information about the number:

If the number is known:

  • SPAM score: Number of negative ratings
  • Categories: Type of SPAM (advertising, fraud, etc.)
  • Comments: Testimonials from other users
  • Reviews: Positive and negative reports

You can rate the number directly as SPAM or legitimate or add it to your blacklist/whitelist.

If the number is unknown:

  • The app indicates that no information is available
  • You can be the first to rate the number
  • Optionally add to blacklist or whitelist

Tip: If you have received a suspicious call from an unknown number, you can look it up here and report it as SPAM.

Search tips

Format of the telephone number

For best search results enter the number with country code:

  • Germany: +49 or 0049 before the number (e.g. +49 30 12345678)
  • Switzerland: +41 or 0041 before the number
  • Austria: +43 or 0043 before the number

The search also works without a country code, but the results are then less reliable.

Data protection during the search

During the search, the number is transmitted to the PhoneBlock server as a SHA-1 hash. The server cannot recalculate the original number from this.

This means: PhoneBlock only finds out which number you are searching for if it already exists in the SPAM database. For all other numbers, your search remains private.

When is the search useful?

Frequent use cases:

  • Check the unknown number on the display before you answer the phone
  • Look up missed calls
  • Check numbers from SMS or e-mails
  • Check whether the number is legitimate before calling back

What you can do after the search:

  • Rate number as SPAM or legitimate
  • Add to blacklist (always block)
  • Add to whitelist (never block)
  • Read comments from other users
