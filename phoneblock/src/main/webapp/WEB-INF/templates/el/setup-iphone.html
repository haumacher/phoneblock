Εγκατάσταση iPhone

Η εγκατάσταση προσθέτει το βιβλίο διευθύνσεων του PhoneBlock στις επαφές σας. Εάν λάβετε μια κλήση από έναν αριθμό ανεπιθύμητης αλληλογραφίας, αυτό θα εμφανίζεται αμέσως, π.χ. "SPAM: 03016637169". Μπορείτε επίσης να δείτε αμέσως στις αναπάντητες κλήσεις ότι δεν αξίζει να καλέσετε ξανά, χωρίς να χρειαστεί να γκουγκλάρετε πρώτα τον αριθμό.

Η εγκατάσταση εκτελείται με τα ακόλουθα βήματα:

  1. Εγγραφείτε στο PhoneBlock.
  2. Ανοίξτε τις "Ρυθμίσεις" > "Επαφές" > "Λογαριασμοί".
  3. Πατήστε "Προσθήκη λογαριασμού" - "Άλλα" > "Προσθήκη λογαριασμού CardDAV".
  4. Εισάγετε τα δεδομένα πρόσβασης και πατήστε "Συνέχεια".
    Διακομιστής
    https://phoneblock.net/contacts/
    Όνομα χρήστη
    Συνδεθείτε για να δείτε το όνομα χρήστη σας.
    κωδικός πρόσβασης
    Εμφανίστηκε σε εσάς αμέσως μετά την πρώτη σύνδεση, μπορείτε να δημιουργήσετε ένα νέο στις ρυθμίσεις.
    Περιγραφή του
    PhoneBlock
  5. Ενεργοποιήστε την επιλογή "Επαφές" και πατήστε "Αποθήκευση".
  6. Τώρα θα βλέπετε τις κλήσεις με "SPAM: ..." στο βιβλίο επαφών σας και όταν χτυπούν. Μπορείτε έτσι να δείτε αμέσως αν αξίζει να απαντήσετε ή να καλέσετε πίσω.

Δεν έχετε πλέον τα δεδομένα πρόσβασής σας στο χέρι; Δεν πειράζει, απλά επαναφέρετε τον κωδικό πρόσβασης στις ρυθμίσεις.

Μπορείτε επίσης να βρείτε οδηγίες με εικόνες αλλού. Σε αυτόν τον οδηγό, χρειάζεται να χρησιμοποιήσετε τα δεδομένα πρόσβασης PhoneBlock μόνο στο βήμα όπου εισάγονται ο διακομιστής, το όνομα χρήστη και ο κωδικός πρόσβασης (βλ. παραπάνω).

Εάν χρησιμοποιείτε το PhoneBlock μόνο στο iPhone, μπορείτε επίσης να αυξήσετε τον αριθμό των καταχωρήσεων στις ρυθμίσεις. Εάν υπάρχουν προβλήματα με αυτό, παρακαλούμε ενημερώστε μας (βλ. FAQ).
