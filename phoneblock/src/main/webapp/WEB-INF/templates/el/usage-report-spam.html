Αναφέρετε αριθμούς ως SPAM ή νόμιμους

Τα μηνύματά σας βοηθούν ολόκληρη την κοινότητα του PhoneBlock να προστατευτεί από τις κλήσεις SPAM.

Αναφορά αριθμού SPAM

Εάν λάβετε μια κλήση SPAM, μπορείτε να αναφέρετε τον αριθμό απευθείας στην εφαρμογή. Όσο περισσότερες λεπτομέρειες παρέχετε, τόσο πιο χρήσιμη θα είναι η αναφορά σας για άλλους χρήστες.

Βήμα 1: Αναφορά SPAM

Πατήστε το κουμπί "Report SPAM" για την αντίστοιχη κλήση στο ιστορικό.

Βήμα 2: Επιλέξτε κατηγορία

Επιλέξτε την κατάλληλη κατηγορία από τη λίστα

Βήμα 3: Σχόλιο (προαιρετικό)

Προαιρετικά προσθέστε ένα σχόλιο για να βοηθήσετε άλλους χρήστες

Διαθέσιμες κατηγορίες SPAM

  • Διαφήμιση: Ανεπιθύμητες διαφημιστικές κλήσεις
  • Διαγωνισμός: Λανθασμένες κοινοποιήσεις βραβείων
  • Παγίδα κόστους: Ακριβοί αριθμοί υψηλής χρέωσης
  • Απάτες: Απατηλές κλήσεις (π.χ. απάτες με εγγόνια)
  • Κλήση Ping: Σύντομο κουδούνισμα για την πρόκληση επανάκλησης
  • Έρευνα: Ανεπιθύμητες έρευνες
  • Άλλο: Άλλοι τύποι SPAM

Χαρακτηρίστε τον αριθμό ως νόμιμο

Εάν ένας αριθμός έχει επισημανθεί εσφαλμένα ως SPAM, μπορείτε να τον αξιολογήσετε ως νόμιμο. Αυτό βοηθά στη διόρθωση λανθασμένων μηνυμάτων και στη βελτίωση της ποιότητας της βάσης δεδομένων.

Χαρακτηρίστε ως νόμιμο

Πατήστε το "Mark as legitimate" για να δώσετε θετική αξιολόγηση. Προαιρετικά, μπορείτε επίσης να προσθέσετε ένα σχόλιο εδώ.

Πότε πρέπει να το κάνω αυτό;

Χαρακτηρίστε έναν αριθμό ως νόμιμο εάν:

  • Πρόκειται για ένα σοβαρό κάλεσμα
  • Ο αριθμός προέρχεται από μια πραγματική εταιρεία
  • Γνωρίζετε τον αριθμό και αναφέρθηκε λανθασμένα
  • Η κλήση ζητήθηκε (π.χ. ιατρείο γιατρού, έμπορος, υπηρεσία δεμάτων)

Σημείωση: Οι νόμιμες αξιολογήσεις βοηθούν στην αφαίρεση του αριθμού από τη βάση δεδομένων SPAM ή στη μείωση της βαθμολογίας SPAM.

Σας ευχαριστώ για τη βοήθειά σας!

Κάθε αναφορά συμβάλλει στη διατήρηση της βάσης δεδομένων του PhoneBlock ενημερωμένης και αξιόπιστης. Η κοινότητα επωφελείται από την εμπειρία σας.
