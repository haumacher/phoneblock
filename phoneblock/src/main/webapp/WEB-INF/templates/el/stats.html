Στατιστικά στοιχεία PhoneBlock

Εδώ μπορείτε να δείτε πώς εξελίσσεται η κοινότητα του PhoneBlock.

Ανάπτυξη χρηστών τις τελευταίες 30 ημέρες

		{
			"type": "line",
			"data": {
				"labels": ,
				"datasets": 
			},
			"options": {
				"responsive": true,
				"maintainAspectRatio": false,
				"scales": {
					"y": {
						"ticks": {
							"precision": 0
						}
					}
				}
			}
		}

Σήμερα: 0 χρήστες συνολικά, +0 από χθες.

Ενεργές χρήσεις τις τελευταίες 30 ημέρες

		{
			"type": "line",
			"data": {
				"labels": ,
				"datasets":
