Flashing af donglen i browseren

Et ESP32-board, et USB-kabel, et klik - den aktuelle PhoneBlock-firmware overføres direkte fra denne browser til din dongle.

Din browser understøtter ikke webseriel Webflashing kører kun i desktop Chrome, Edge eller Opera. På Firefox, Safari, Android og iOS kan du flashe donglen med ESP-IDF-værktøjskæden via USB i stedet.

1. Forbered hardware

  • ESP32-DevKit med ESP-WROOM-32E (f.eks. NodeMCU-32E) - se Hardwarenoter.
  • USB-datakabel (ikke bare et opladningskabel) mellem computer og dongle.
  • På Windows kan det være nødvendigt at installere den serielle USB-driver - normalt CP210x (Silicon Labs) eller CH340/CH341 (WCH).
  • På Linux skal din bruger være i gruppen dialout (sudo usermod -aG dialout $USER, log derefter ud og ind igen en gang).

2. Blinkende

Klik på knappen, vælg din dongle i browserdialogen, og vent, indtil processen er færdig. Flash-hukommelsen slettes under en første flash - enhver eksisterende konfiguration vil gå tabt.

3. Efter blinkning

  1. Donglen genstarter og åbner WLAN PhoneBlock-Dongle-XXXX.
  2. Skift til dette WLAN med din smartphone eller notebook, så åbner browseren automatisk opsætningssiden (captive portal).
  3. Vælg hjemme-WLAN, indtast SIP-adgangsdata og PhoneBlock-token - se detaljer i Dongle-instruktioner.

Kildekode til firmwaren: phoneblock-dongle på GitHub.
