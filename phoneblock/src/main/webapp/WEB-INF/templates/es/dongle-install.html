Flashear el dongle en el navegador

Una placa ESP32, un cable USB, un clic - el último firmware PhoneBlock se transfiere directamente desde este navegador a su dongle.

Tu navegador no soporta web serial El flasheo web solo funciona en Chrome, Edge u Opera de escritorio. En Firefox, Safari, Android e iOS, puede flashear el dongle con el ESP-IDF toolchain via USB en su lugar.

1. preparar el hardware

  • ESP32-DevKit con ESP-WROOM-32E (por ejemplo NodeMCU-32E) - ver Notas de hardware.
  • USB-cable de datos (no sólo un cable de carga) entre el ordenador y el dongle.
  • En Windows puede que necesite instalar el controlador serie USB - normalmente CP210x (Silicon Labs) o CH340/CH341 (WCH).
  • En Linux, su usuario debe estar en el grupo dialout (sudo usermod -aG dialout $USER, luego cierre la sesión y vuelva a entrar una vez).

2. parpadeo

Haz clic en el botón, selecciona tu dongle en el cuadro de diálogo del navegador y espera a que finalice el proceso. La memoria flash se borra durante un flasheo inicial - cualquier configuración existente se perderá.

3. después del flasheo

  1. El dongle se reinicia y abre la WLAN PhoneBlock-Dongle-XXXX.
  2. Conéctese a esta WLAN con su smartphone o portátil, el navegador abrirá automáticamente la página de configuración (portal cautivo).
  3. Seleccione la WLAN doméstica, introduzca los datos de acceso SIP y el token PhoneBlock - detalles en las Instrucciones de la mochila.

Código fuente del firmware: phoneblock-dongle en GitHub.
