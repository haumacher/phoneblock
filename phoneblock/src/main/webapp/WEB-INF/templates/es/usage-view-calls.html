Ver llamadas bloqueadas y perdidas

PhoneBlock Mobile comprueba todas las llamadas entrantes de números desconocidos y le muestra cuáles han sido bloqueadas y cuáles han pasado.

Importante: ¡Sólo se comprueban los números desconocidos!

Las llamadas de números de tu lista de contactos nunca se comprueban y no aparecen en la lista de llamadas de PhoneBlock. La aplicación solo procesa los números que no están guardados en la agenda.

Llamadas SPAM bloqueadas

Las llamadas de SPAM bloqueadas se muestran con un símbolo rojo y una puntuación de SPAM. Recibirá una notificación sobre el bloqueo.

¿Qué números se bloquean como SPAM?

Números SPAM (serán bloqueados)

Un número se bloquea como SPAM si:

  • tiene una puntuación alta de SPAM en la base de datos de PhoneBlock
  • La puntuación de SPAM supera el valor umbral establecido
  • Ella está en su lista negra personal (independientemente de la puntuación)

Resultado: Su teléfono no suena, la llamada se rechaza automáticamente.

Números legítimos (se les permite el paso)

Un número se considera legítimo si:

  • No está disponible en la base de datos PhoneBlock
  • Tiene una puntuación baja de SPAM por debajo de su umbral
  • Está en tu lista blanca personal (nunca será bloqueada)

Resultado: Tu teléfono suena con normalidad, la llamada pasa.

¿Cómo funciona el bloqueo?

PhoneBlock Mobile utiliza el CallScreeningService de Android para filtrar las llamadas entrantes antes de que suenen. Si el número está en la base de datos de SPAM y se supera el umbral establecido, la llamada se bloquea automáticamente.

¿Cuándo hay que comunicar un número?

  • El número aún no está en la base de datos de PhoneBlock (puntuación de SPAM baja o nula)
  • La llamada era claramente no deseada (publicidad, fraude, etc.)
  • Usted quiere advertir a la comunidad acerca de este número
  • El número se bloqueará automáticamente en el futuro

¿Qué ocurre en caso de denuncia?

  • El número se añade automáticamente a tu lista negra</x1
  • Estarás protegido de futuras llamadas desde este número
  • Tu mensaje contribuye a la base de datos PhoneBlock
  • Otros usuarios se benefician de tu valoración

A las instrucciones: Denunciar números de SPAM

Importante:

Si denuncias un número como SPAM, se añadirá a tu lista negra personal y se bloqueará siempre en el futuro. Esta decisión tiene prioridad sobre la valoración de la comunidad.
