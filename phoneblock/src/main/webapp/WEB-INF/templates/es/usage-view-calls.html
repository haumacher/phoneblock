Ver llamadas bloqueadas y perdidas

PhoneBlock Mobile le muestra todas las llamadas SPAM bloqueadas y las llamadas perdidas de números desconocidos.

Protección de datos: Las llamadas de los números de tu lista de contactos no se graban. La aplicación solo procesa números desconocidos.

Cómo funciona

Bloqueo automático

Las llamadas SPAM bloqueadas se muestran automáticamente en la aplicación. Recibirás una notificación con el número y la puntuación de SPAM.

Detalles de la llamada

Pulse sobre una entrada del historial de llamadas para ver los detalles:

  • Puntuación SPAM: Número de valoraciones negativas de la comunidad.
  • Hora: Cuando se realizó la llamada.
  • Categorías: Tipo de SPAM (por ejemplo, publicidad, fraude, competencia)
  • Comentarios: Informes y experiencias de otros usuarios de PhoneBlock.

Desde aquí también puedes calificar tú mismo el número o añadirlo a tu lista negra/blanca personal.

¿Cómo funciona el bloqueo?

PhoneBlock Mobile utiliza el CallScreeningService de Android para filtrar las llamadas entrantes antes de que suenen. Si el número está en la base de datos de SPAM y se supera el umbral establecido, la llamada se bloquea automáticamente.
