Gestión de listas negras y blancas personales

Además de la base de datos comunitaria, puede mantener sus propias listas para personalizar el filtrado de llamadas.

¿Qué son la lista negra y la lista blanca?

Lista negra

Los números de tu lista negra están siempre bloqueados, independientemente de si son conocidos como SPAM en la base de datos de PhoneBlock o no.

Uso: Para llamadas molestas que desea bloquear personalmente.

Lista blanca

Los números de tu lista blanca nunca serán bloqueados, incluso si son reportados como SPAM en la base de datos de PhoneBlock.

Uso: Para números importantes que podrían bloquearse por error.

Gestionar listas

Paso 1: Abrir la configuración

Toca el icono de menú (☰) en la parte superior izquierda y selecciona "Ajustes"

Paso 2: Seleccionar lista

Seleccione "Lista negra" o "Lista blanca" en la configuración

Paso 3: Mostrar lista

Aparecen tus números personales

Añadir número

Para añadir un nuevo número a su lista negra o blanca, pulse el símbolo más (+) en la parte superior derecha de la lista correspondiente. A continuación, introduce el número de teléfono (con el prefijo del país para obtener mejores resultados).

Eliminar número de la lista

Desliza el dedo hacia la izquierda sobre el número y pulsa el icono de la papelera para eliminarlo.

Sincronización:

Sus listas personales se sincronizan con su cuenta PhoneBlock y están disponibles automáticamente en todos sus dispositivos en los que haya iniciado sesión con la misma cuenta.

Consejos de uso

Lista negra

  • Añada los números que le molestan personalmente
  • Aunque aún no estén en la base de datos de PhoneBlock
  • El número se bloquea inmediatamente
  • Ideal para llamantes persistentes

Lista blanca

  • Añadir números importantes (médico, colegio, trabajo)
  • Evita el bloqueo accidental
  • Tiene prioridad sobre la base de datos comunitaria
  • Especialmente importante para las llamadas críticas

Nota: Tus listas personales siempre tienen prioridad sobre la base de datos de la comunidad PhoneBlock. Un número en su lista blanca nunca será bloqueado, incluso si ha sido reportado como SPAM por miles de otros usuarios.
