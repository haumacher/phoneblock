Buscar números

Puede buscar cualquier número de teléfono en la base de datos PhoneBlock, incluso sin que se haya producido una llamada.

Cómo buscar un número

Paso 1: Abrir la búsqueda

Pulse el icono de la lupa en la parte superior derecha para abrir la función de búsqueda.

Paso 2: Introducir el número

Introduzca el número de teléfono y pulse "Buscar".

Lo que muestra la búsqueda

Tras la búsqueda verás la siguiente información sobre el número:

Si se conoce el número:

  • Puntuación SPAM: Número de valoraciones negativas
  • Categorías: Tipo de SPAM (publicidad, fraude, etc.)
  • Comentarios: Testimonios de otros usuarios
  • Revisiones: Informes positivos y negativos

Qué puede hacer después de la búsqueda:

  • Califique el número como SPAM o legítimo
  • Añadir a la lista negra (bloquear siempre)

Si el número es desconocido:

  • La aplicación indica que no hay información disponible
  • Añadir a lista blanca (nunca bloquear)

Leer comentarios de otros usuarios

  • Una clasificación SPAM añade el número a su lista negra.
  • Una calificación legítima añade el número a tu lista blanca.

Consejo: Si has recibido una llamada sospechosa de un número desconocido, puedes buscarla aquí y denunciarla como SPAM.

Consejos de búsqueda

Formato del número de teléfono

Puede introducir números con o sin indicativo de país:

  • Con el prefijo del país: +49 30 12345678 (Alemania), +41 44 1234567 (Suiza), +43 1 1234567 (Austria)
  • Sin código de país: El número se complementa automáticamente con el código de país de su configuración.

Consejo: Si buscas un número extranjero, introduce siempre el prefijo completo del país.

Protección de datos durante la búsqueda

Durante la búsqueda, el número se transmite al servidor de PhoneBlock en texto plano. Esto es necesario para poder calificar el número y añadirlo a la base de datos.

Importante: La función de protección de datos basada en hash (SHA-1) sólo se utiliza para las llamadas entrantes. Para la búsqueda activa, se requiere la transmisión completa del número para habilitar todas las funciones (valoración, comentarios).

¿Cuándo es útil la búsqueda?

Casos de uso frecuentes:

  • Compruebe el número desconocido en la pantalla antes de contestar al teléfono
  • Buscar llamadas perdidas
  • Comprobar números de SMS o correos electrónicos
  • Compruebe si el número es legítimo antes de devolver la llamada

Qué puede hacer después de la búsqueda:

  • Calificación como SPAM (se añadirá a la lista negra → siempre bloqueado)
  • Calificación como legítimo (se añadirá a la lista blanca → nunca se bloqueará)
  • Leer comentarios de otros usuarios
  • Si lo desea, añada su propio comentario

Nota: ¡Calificar y añadir a la lista es una misma acción!
