Notificar números como SPAM o legítimos

Tus mensajes ayudan a toda la comunidad PhoneBlock a protegerse de las llamadas SPAM.

Informar del número de SPAM

Si recibes una llamada SPAM, puedes denunciar el número directamente en la aplicación. Cuantos más detalles proporciones, más útil será tu denuncia para otros usuarios.

Paso 1: Informar sobre SPAM

Pulse sobre el botón "Denunciar SPAM" de la llamada correspondiente en el historial

Paso 2: Seleccionar categoría

Seleccione la categoría adecuada de la lista

Paso 3: Comentar (opcional)

Si lo desea, puede añadir un comentario para ayudar a otros usuarios

Categorías de SPAM disponibles

  • Publicidad: Llamadas publicitarias no solicitadas
  • Concurso: Notificaciones de premios incorrectas.
  • Trampa de costes: Números de tarificación adicional caros
  • Estafas: Llamadas fraudulentas (por ejemplo, estafas de nietos)
  • Ping de llamada: Timbre corto para provocar una devolución de llamada.
  • Encuesta: Encuestas no deseadas
  • Otros: Otros tipos de SPAM

Marcar número como legítimo

Si un número ha sido marcado incorrectamente como SPAM, puedes calificarlo como legítimo. Esto ayuda a corregir los mensajes incorrectos y a mejorar la calidad de la base de datos.

Marcar como legítimo

Pulse sobre "Marcar como legítimo" para dar una valoración positiva. Opcionalmente, también puede añadir un comentario aquí.

¿Cuándo debo hacerlo?

Marca un número como legítimo si:

  • Es un llamamiento serio
  • El número procede de una empresa real
  • Conoces el número y se informó incorrectamente
  • La llamada fue solicitada (por ejemplo, consulta médica, comerciante, servicio de paquetería)

Nota: Las valoraciones legítimas ayudan a eliminar el número de la base de datos de SPAM o a reducir la puntuación de SPAM.

Gracias por su ayuda.

Cada informe ayuda a mantener la base de datos PhoneBlock actualizada y fiable. La comunidad se beneficia de tu experiencia.
