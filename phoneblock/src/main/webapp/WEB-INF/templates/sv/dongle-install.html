Flasha dongeln i webbläsaren

Ett ESP32-kort, en USB-kabel, ett klick - den senaste inbyggda programvaran för PhoneBlock överförs direkt från webbläsaren till din dongel.

Din webbläsare stöder inte web serial Web flashing körs endast i desktop Chrome, Edge eller Opera. På Firefox, Safari, Android och iOS kan du flasha dongeln med ESP-IDF toolchain via USB istället.

1. Förbered hårdvara

  • ESP32-DevKit med ESP-WROOM-32E (t.ex. NodeMCU-32E) - se Hårdvarunoter.
  • USB-datakabel (inte bara en laddningskabel) mellan dator och dongel.
  • På Windows kan du behöva installera den seriella USB-drivrutinen - vanligtvis CP210x (Silicon Labs) eller CH340/CH341 (WCH).
  • På Linux måste din användare vara med i gruppen dialout (sudo usermod -aG dialout $USER, logga sedan ut och in igen en gång).

2. Blinkande

Klicka på knappen, välj din dongle i webbläsarens dialogruta och vänta tills processen är klar. Flashminnet raderas under en första flash - eventuell befintlig konfiguration går förlorad.

3. efter blinkning

  1. Dongeln startar om och öppnar WLAN PhoneBlock-Dongle-XXXX.
  2. Växla till detta WLAN med din smartphone eller bärbara dator, webbläsaren öppnar installationssidan automatiskt (captive portal).
  3. Välj hem-WLAN, ange SIP-åtkomstdata och PhoneBlock-token - mer information finns i instruktionerna för Dongle.

Källkod för den inbyggda programvaran: phoneblock-dongle på GitHub.
