Hantera personlig svart- och vitlista

Förutom community-databasen kan du underhålla dina egna listor för att anpassa samtalsfiltreringen.

Vad är svartlista och vitlista?

Svarta listan

Numren på din svarta lista är alltid blockerade, oavsett om de är kända som SPAM i PhoneBlock-databasen eller inte.

Användning: För irriterande uppringare som du vill blockera personligen.

Vitlista

Numren på din vitlista kommer aldrig att blockeras, även om de rapporteras som SPAM i PhoneBlock-databasen.

Användning: För viktiga nummer som kan blockeras av misstag.

Hantera listor

Steg 1: Öppna inställningar

Tryck på menyikonen (☰) längst upp till vänster och välj "Inställningar"

Steg 2: Välj lista

Välj "Blacklist" eller "Whitelist" från inställningarna

Steg 3: Visa listan

Dina personliga nummer visas

Lägg till nummer

Om du vill lägga till ett nytt nummer i din svarta eller vita lista trycker du på plus-symbolen (+) längst upp till höger i respektive lista. Ange sedan telefonnumret (med landskod för bästa resultat).

Ta bort nummer från listan

Svep åt vänster över numret och tryck på papperskorgsikonen för att ta bort det

Synkronisering:

Dina personliga listor synkroniseras med ditt PhoneBlock-konto och är automatiskt tillgängliga på alla dina enheter som du är inloggad på med samma konto.

Tips för användning

Svarta listan

  • Lägg till siffror som stör dig personligen
  • Även om de ännu inte finns i PhoneBlock-databasen
  • Numret blockeras omedelbart
  • Idealisk för ihärdiga uppringare

Vitlista

  • Lägg till viktiga nummer (läkare, skola, arbete)
  • Förhindrar oavsiktlig blockering
  • Har prioritet över community-databasen
  • Särskilt viktigt vid kritiska samtal

Anmärkning: Dina personliga listor har alltid företräde framför PhoneBlocks community-databas. Ett nummer på din vitlista kommer aldrig att blockeras, även om det har rapporterats som SPAM av tusentals andra användare.
