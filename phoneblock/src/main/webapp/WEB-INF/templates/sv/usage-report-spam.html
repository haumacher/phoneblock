Rapportera nummer som SPAM eller legitima

Dina meddelanden hjälper hela PhoneBlock-gruppen att skydda sig mot SPAM-samtal.

Rapportera SPAM-nummer

Om du får ett SPAM-samtal kan du rapportera numret direkt i appen. Ju mer information du ger, desto mer användbar blir din rapport för andra användare.

Steg 1: Rapportera SPAM

Tryck på knappen "Rapportera SPAM" för motsvarande samtal i historiken

Steg 2: Välj kategori

Välj lämplig kategori från listan

Steg 3: Kommentera (valfritt)

Lägg eventuellt till en kommentar för att hjälpa andra användare

Tillgängliga SPAM-kategorier

  • Reklam: Obeställda reklamsamtal
  • Tävling: Felaktiga prismeddelanden
  • Kostnadsfälla: Dyra premiumnummer
  • Bedrägerier: Bedrägliga samtal (t.ex. barnbarnsbedrägerier)
  • Ping call: Kort ringsignal för att provocera fram ett återuppringningssamtal
  • Enkät: Oönskade enkäter
  • Andra: Andra typer av SPAM

Markera numret som legitimt

Om ett nummer felaktigt har markerats som SPAM kan du bedöma det som legitimt. Detta hjälper till att korrigera felaktiga meddelanden och förbättra kvaliteten på databasen.

Markera som legitim

Tryck på "Markera som legitim" för att ge ett positivt omdöme. Alternativt kan du också lägga till en kommentar här.

När ska jag göra det här?

Markera ett nummer som legitimt om:

  • Det är ett allvarligt samtal
  • Numret kommer från ett riktigt företag
  • Du känner till siffran och den rapporterades felaktigt
  • Samtalet efterfrågades (t.ex. läkarmottagning, hantverkare, paketutlämning)

Anmärkning: Legitima betyg hjälper till att ta bort numret från SPAM-databasen eller sänka SPAM-poängen.

Tack för din hjälp!

Varje rapport bidrar till att hålla PhoneBlock-databasen uppdaterad och tillförlitlig. Gemenskapen drar nytta av din erfarenhet.
